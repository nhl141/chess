# How to use 
- The alg_engines folder includes, as the name suggests, chess engines based on algorithmic search.
- The Pretrained-models-test folder includes pretrained engines I imported.
- The lcc0-mcts-engine folder includes the lc0 RL pretrained model with an integrated Monte-Carlos Tree Search Code.
